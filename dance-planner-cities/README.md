# Cities

## Usage
import { cities } from 'dance-planner-cities'

console.log(cities[0]);

This should write:
```
{
  countryName: 'Andorra',
  countryCode: 'AD',
  cities: [
    { name: 'les Escaldes', lat: 1.53414, lon: 42.50729 },
    { name: 'Encamp', lat: 1.58014, lon: 42.53474 },
    { name: 'Andorra la Vella', lat: 1.52109, lon: 42.50779 }
  ]
}
```
